# Academic and Theory Reports

This directory contains selected academic reports and thesis-style writing. These projects are useful for demonstrating mathematical depth, algorithmic thinking, and technical writing ability.

## Files

- `masters-thesis-uniform-k-client-problem.pdf` — M.Sc. thesis on the uniform k-client problem, an online-algorithms problem dual to k-server-style models.
- `potential-function-proofs-gradient-methods.pdf` — survey-style project on potential-function proofs for gradient descent, mirror descent, Frank-Wolfe, and Nesterov acceleration.
- `decision-making-under-uncertainty-prophet-philosopher-inequalities.pdf` — seminar project on online Bayesian bipartite matching, prophet inequalities, and a philosopher/ski-rental direction.
- `bsc-sonar-object-classification.pdf` — B.Sc. project on classifying objects from sonar signals using machine-learning methods.
